## 簡介
* 利用 input 元素的 checkbox 來實作不同狀態下的排版，配合使用 css 的 :checked，來達成不同效果。在當 :checked 時，再利用 ~ 的 selector 將後方所有元素制定狀態！
* 程式中盡量利用 css 處理，非不得已或功能需求才使用 JavaScript 控制顯示。
* 更多生活地圖請[看這裡](https://livemaps.ioa.tw/)。
